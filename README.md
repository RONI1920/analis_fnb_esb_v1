# 📈 Stock Peer Group Analysis

An interactive Streamlit dashboard to compare how different stocks perform against their industry peers using financial metrics, market performance, and visual analytics.

Built with Python, Streamlit, and modern data visualization libraries.

---

# 🚀 Live Demo

Try the application instantly:

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://stockpeers.streamlit.app)

---

# ✨ Features

- 📊 Compare multiple stocks side-by-side
- 📈 Interactive performance charts
- 🏢 Peer group analysis by sector/industry
- 💹 Financial metric visualization
- 🔍 Easy stock search and filtering
- ⚡ Fast and responsive Streamlit interface
- 📱 Clean and modern dashboard UI

# 📦 Installation

## 1. Clone the repository

```bash
git clone https://github.com/streamlit/demo-stockpeers.git
cd demo-stockpeers
```

---

## 2. Create virtual environment

Using `uv`:

```bash
uv venv
```

Activate the environment:

### Windows

```bash
.venv\Scripts\activate
```

### Linux / macOS

```bash
source .venv/bin/activate
```

---

## 3. Install dependencies

```bash
uv sync
```

---

# ▶️ Run the App

Start the Streamlit server:

```bash
streamlit run app.py
```

The app will be available at:

```text
http://localhost:8501
```
